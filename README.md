# SetSpawn Plugin

Sunucularınız İçin Geliştirilmiş **Basit** Bir Spawn Sistemi.  
Spawn Ayarlama, Spawn’a Işınlanma ve Oyuncular Sunucuya Girdiğinde Otomatik Spawn’a Gönderme Özelliklerini İçerir.
## ✨ Özellikler
- 📍 `/setspawn` İle Bulunduğun Konumu Spawn Olarak Ayarlama
- 🚀 `/spawn` Komutu İle Spawn’a Işınlanma
- 🔄 Oyuncu Sunucuya Girdiğinde Otomatik Spawn Bölgesine Işınlanır
- 💾 Spawn Bilgileri `config.yml` İçinde Saklanır
- 🔐 Permission Sistemi
- ⚡ Hafif ve Performans Dostu
## 📦 Gereksinimler
- **Spigot / Paper** (1.13+ önerilir)
- Ekstra Bağımlılık **yoktur**
## ⚙️ Kurulum
1. `SetSpawn.jar` Dosyasını `plugins/` Klasörüne At
2. Sunucuyu Başlat veya Restart At
3. `config.yml` Otomatik Oluşur
4. `/setspawn` İle Spawn Belirle
5. Hazır 🎉
## 🧾 Komutlar
| Komut | Açıklama |
|------|---------|
| `/setspawn` | Bulunduğun Konumu Spawn Bölgesi Olarak Ayarlar |
| `/spawn` | Spawn Noktasına Işınlanır |
## 🔑 Yetkiler
| Permission | Açıklama |
|-----------|---------|
| `setspawn.use` | `/setspawn` Komutunu Kullanma |
> `/spawn` Komutu İçin Permission Gerekmez"
